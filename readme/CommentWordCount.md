## CommentWordCount
### Description
Counts the words from `Text` field.

### Input
    <row Id="1" 
    PostId="7" 
    Score="2" 
    Text="Just ask this @Robert how annoying this can be..." 
    CreationDate="2010-06-30T19:29:03.500" 
    UserId="17" />
### Output
    just 1
    ask 1
    robert 1
    how 1
    annoying 1
    this 2
    can 1
    be 1
