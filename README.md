MapReduce
=========

Hadoop Map-Reduce Design Patterns

How to use the MapReduce:
--------------------------

Clone the repository:
```
git clone git@github.com:geftimov/MapReduce.git
```
Go in to the folder:
```
cd MapReduce
```
Build it with Maven:
```
mvn clean install
```
Example run in each individual pattern example.
